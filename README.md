# bionemo-aspirin-optimization
BioNeMo Aspirin Optimization  Molecular optimization of Aspirin using NVIDIA BioNeMo's MolMIM model (CMA-ES algorithm) to maximize QED.  Validated via SwissADME. The top candidate achieved a 0.939 QED, 85% bioavailability, BBB permeation, and excellent synthetic accessibility (1.99). See raw data in `resultados.json`.
